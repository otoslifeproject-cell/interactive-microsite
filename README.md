# OTOS Continuity Microsite — Static Depth + Glass + Stat Motion Final

Fresh root-files build for GitHub/Vercel upload.

## Upload root files
Upload these files directly at the repository root:

- `index.html`
- `styles.css`
- `script.js`
- `README.md`
- `assets/`
- `VERSION.txt`

Do not upload a wrapper folder as the project root.

## Build marker
Search GitHub or view-source for:

`OTOS_STATIC_DEPTH_STAT_MOTION_FINAL_20260627`

If that marker is present in live source, this build is deployed.

## Included visual/UX work

- Five-module PMOS-style education-series structure.
- Static fixed atmosphere layer behind reader content.
- Deeper image-like background layers: paper, line field, lens, cocoon, dual field and route.
- Warm editorial glass panels scrolling over the static atmosphere.
- Module-specific background states for Modules 01–05.
- Module 02 medication atmosphere: dark-to-warm amber panel, document form and stable amber thread.
- Module 03 ADHD Lens visual state, non-surveillance.
- Module 04 Dual Screen recognition frame.
- Module 05 integrated practice with cocoon, route advice, governed test, hold-back and close.
- Restrained stat-settle animation for `61.3%`, `50`, `12`, and `C&P`.
- Reduced-motion support.
- Mobile overflow and blur/performance guards.
